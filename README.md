Requisitos funcionales Rol Administrador

Identificador del requerimiento
RF-002
Nombre del requerimiento
Cambio de roles de usuarios
Características
El sistema permitirá al administrador modificar los roles de los usuarios (entrenador, atleta y pasante).
Descripción
El administrador del sistema podrá asignar, cambiar o remover roles a los usuarios registrados.
Prioridad 
ALTA



Identificador del requerimiento
RF-003
Nombre del requerimiento
Registro de usuarios
Características
El sistema debe permitir al administrador, registrar, los datos del usuario
Descripción
El sistema permitirá registrar a los nuevos usuarios (identificación, apellidos, nombres, dirección, teléfono, correo electrónico, tipo de usuario [entrenador, pasante])
Prioridad 
ALTA

Características

Descripción

RF-005

Listado de usuarios
